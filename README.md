# Vinyl Collection

A web-based vinyl record collection viewer showcasing 311 classic rock and metal albums with artwork from iTunes.

## Features

- Browse a curated collection of 311 vinyl albums
- View high-quality album artwork
- Search and filter albums by artist or title
- Responsive design using Tailwind CSS
- Interactive React-based UI with smooth animations

## Technologies Used

- **React 18** - UI framework
- **Tailwind CSS** - Styling
- **Babel Standalone** - JSX transformation
- **iTunes API** - Album artwork

## Usage

Simply open `index.html` in a web browser to view the collection. No build process or dependencies required - all resources are loaded via CDN.
